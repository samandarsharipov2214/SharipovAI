# Financial Knowledge Library

SharipovAI получил библиотеку финансового обучения для внутренних AI-ботов.

## Главная идея

Боты должны понимать:

```text
криптовалюты
акции
биржи
покупку и продажу
риск
финансовые институты
макроэкономику
регулирование
```

Но система не должна незаконно хранить полные тексты чужих книг, платных статей или закрытых материалов.

Правильный подход:

```text
источник -> конспект -> правило -> проверка -> учебный пакет бота
```

## Что реализовано

Файл:

```text
learning/financial_knowledge_library.py
```

Он содержит:

```text
домены знаний
типы источников
карту бот -> темы
реестр учебных источников
ключевые понятия
минимальные правила
curriculum для каждого бота
```

## Домены

```text
crypto
stocks
exchanges
trading
financial_institutions
macro
risk
regulation
```

## API

Файл:

```text
learning/learning_app.py
```

Новые endpoints:

```text
GET /api/learning/finance/manifest
GET /api/learning/finance/bots/{bot_name}
```

## Что получает бот

Например, Risk Engine получает:

```text
risk
trading
exchanges
financial_institutions
```

И правила:

```text
не делать вывод без источника
не рекомендовать сделку без риск-проверки
учитывать стакан, спред, комиссии и проскальзывание
при высоком риске снижать размер позиции или блокировать сделку
```

News Agent получает:

```text
crypto
stocks
macro
regulation
```

И обязан учитывать:

```text
официальные источники
регуляторные новости
подтверждение информации
отделение факта от предположения
```

## Что НЕ реализовано пока

Пока система не скачивает интернет сама и не хранит полные книги.

Следующие шаги:

```text
1. Добавить загрузку пользовательских PDF/книг/статей.
2. Делать из них краткий конспект.
3. Превращать конспект в правила.
4. Добавлять правила в учебные пакеты ботов.
5. Делать экзамен по загруженному материалу.
6. Хранить всё в SQLite.
```

```

```
